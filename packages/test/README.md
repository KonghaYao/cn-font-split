## Usage

1. 先在 subsets 文件夹 build 所有的 test 文件

2. 安装依赖

```bash
$ pnpm install
$ playwright install
```

```bash
$ node ./script/index.mjs
$ pnpm dev
$ pnpm test # 这个测试工具，第一次会生成数据，导致测试失败，再来一次就好了
```
